Font Style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa properti yang dapat digunakan untuk mengatur style teks yaitu :

  • text-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right.
  • color: digunakan untuk memberikan warna pada teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: digunakan mengatur jarak antar kata.
